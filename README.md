# Task CLI

A minimal todo list manager written in pure C.

## Features
- Add task
- View task
- Mark task as complete (and delete them)

## Usage
```bash
task -a "Buy coffee"
task -v 
task -c 2
```

## Installation
```bash
git clone https://github.com/Krish-jswl/task-cli.git && cd task-cli
```

```bash
make
```

```bash
make && sudo mv task /usr/local/bin/
```
